# qtimer
A webapp providing timers for a club event.

### To Setup for Development and Building
```
npm i --save package.json
```

### To Run a Development Server
```shell
npm start
```

### To Make a Production Build
```shell
npm run build
```

### File Structure
```
src
├── App.css (main styling file) 
├── App.js (main js file)
├── App.test.js
├── chime.wav
├── index.css
├── index.js
├── reportWebVitals.js
└── setupTests.js
```
